+ 只需要 Sub-Store 生成 sing-box 格式的节点配置文件（nodes.json）然后和基本配置（sing-box.json）合并 
+ 如果你没有搭建 sub-store 你可以用 mihomo party 自带的 sub-store 生成节点的 json 文件 
+ 脚本用法： `python3 merge.py sing-box.json nodes.json` 会生成一个新的配置文件，带有节点信息
